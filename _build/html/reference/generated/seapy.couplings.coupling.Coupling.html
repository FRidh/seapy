clf[source]¶

Coupling loss factor eta.

Return type:numpy.ndarray

In case the CLF is not specified for the given coupling it is calculated using the SEA consistency relation.

eta_{12} = eta_{21} frac{n_2}{n_1}

conductivity[source]¶
